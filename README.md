# Simple Home & Contact Website

This is a simple multi-page website built using **HTML** and **CSS**. The project consists of a **Home page** and a **Contact page**, designed with a clean and user-friendly interface. It demonstrates the fundamentals of web development, including page structure, navigation, styling, and responsive design principles.

## Features
- Home page with introductory content
- Contact page with contact information/form layout
- Easy navigation between pages
- Clean and responsive design
- Built using only HTML and CSS

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to practice and improve front-end web development skills by building a simple, functional, and visually appealing website.

## Getting Started
1. Clone this repository.
2. Open `index.html` in your web browser.
3. Navigate through the Home and Contact pages.
